# Content-Erstellung Checkliste

> Schnell-Referenz zum Abhaken bei jedem neuen Content-Stück.

## Pre-Flight
- [ ] Thema validiert (Nutzerfrage beantwortbar?)
- [ ] Eigene Erfahrung vorhanden?
- [ ] Primary Keyword identifiziert
- [ ] 3-5 Secondary Keywords
- [ ] 3 Frage-Keywords für FAQ

## Struktur
- [ ] Hero-Block: TL;DR ≤ 50 Wörter
- [ ] Hero-Block: Für wen / Zeitaufwand / Ergebnis
- [ ] Kontext-Block: Eigene Erfahrung erwähnt
- [ ] Hauptteil: Max. 3 Sätze pro Absatz
- [ ] Edge Cases: Min. 2 häufige Fehler
- [ ] FAQ: 3-5 Fragen, je ≤ 60 Wörter
- [ ] Weiterführend: Min. 2 interne Links

## Technik
- [ ] Schema.org Markup (Article/HowTo/FAQPage)
- [ ] URL slug = Primary Keyword
- [ ] Title Tag ≤ 60 Zeichen
- [ ] Meta Description ≤ 155 Zeichen
- [ ] Nur ein H1
- [ ] Alt-Text für alle Bilder
- [ ] Canonical Tag

## KI-Optimierung
- [ ] Information Gain (Fakten, die KI nicht kennt)
- [ ] Quellen belegt
- [ ] Klar extrahierbare Antworten
- [ ] Keine Widersprüche

## Publish
- [ ] Rechtschreibung & Grammatik
- [ ] Alle Links funktionieren
- [ ] Mobile getestet
- [ ] Ladezeit < 3s
- [ ] Google Search Console eingereicht
